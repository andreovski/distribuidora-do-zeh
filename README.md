## Descrição do Projeto

Aplicativo Android desenvolvido em Kotlin para controle de estoque de uma distribuidora de bebidas. O sistema permite o gerenciamento completo de produtos, incluindo cadastro, edição, exclusão, movimentações de estoque (entradas e saídas) e geração de relatórios estatísticos.

## Tecnologias Utilizadas

- **Linguagem**: Kotlin
- **Framework UI**: Jetpack Compose
- **Arquitetura**: MVVM (Model-View-ViewModel)
- **Banco de Dados**: Room (SQLite)
- **Navegação**: Navigation Compose
- **Coroutines**: Para operações assíncronas
- **ViewModel**: Gerenciamento de estado reativo
- **Flow/StateFlow**: Para streams de dados reativos
